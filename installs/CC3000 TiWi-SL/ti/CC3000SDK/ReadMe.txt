===================
CC3000 SDK
===================
Version : 1.14

CC3000 SDK contains Basic Wifi Application for MSP430FR5739 platform.

===================
Contents
===================
This release supports Basic Wifi Application for MSP430FR5739 platform. Under the installation directory there is a directory for MSP430FR5739 platform containing this application. The application contains webpage link showing how to use the application. Application also contains the binary file along with the flashing tool to burn the binary.
